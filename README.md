# crud_proyecto_diego
CRUD de la base de datos del proyecto.

CRUD - Base de datos Reserva Natural
=====================================
Aplicación web en PHP para gestionar la relación
Visitante ↔ Actividad de una reserva natural.

Tecnologías: PHP, MySQL, XAMPP
Tabla principal: Visitante_Realiza_Actividad
  (relación muchos a muchos entre Visitante y Actividad)

Instrucciones de instalación:
1. Instalar XAMPP
2. Importar BASEDEDATOS.sql en phpMyAdmin
3. Copiar la carpeta crud_proyecto en htdocs/
4. Abrir http://localhost/crud_proyecto/visitante_actividad/index.php
